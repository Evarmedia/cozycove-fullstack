# CozyCove Ecormmerce Platform
## Built with 
### FRONTEND
- React + vite
- Tailwind

### BACKEND
- Nodejs
- EXPRESSJS
- MONGODB

# Dependencies used
- npm install axios
- npm install --save react-toastify
- npm install react-router-dom
- npm install react-icons
- npm install @mui/material @emotion/react @emotion/styled
- npm install @mui/icons-material
